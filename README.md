# 저장소 개요
- 본 저장소는 SW Maestro 13기 과정에서 진행하는 '현지야 서비스'의 서버 Application을 관리하는 저장소이다.

## 팀 정보
- SW Maestro 13기 TGT팀  

| 이름  | 역할  |업무|
|-----|-----|---|
| 우태균 | 팀장  |BE|
| 신우진 | 팀원  |FE/BE|
| 최정인 | 팀원  |FE|
